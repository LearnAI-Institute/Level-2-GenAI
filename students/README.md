# Students Directory

Welcome to the **students** folder! This is where all student assignments and work will be submitted.

## Contribution Rules and Instructions

### 📁 Folder Structure

Each student should create a folder with their name in the following format:

```
students/
├── your-name/              # Replace 'your-name' with your actual name (e.g., john-smith)
│   ├── assignment-1/       # Create folders for each assignment
│   ├── assignment-2/
│   ├── project-1/
│   └── README.md           # Optional: Add a personal README about your work
├── another-student/
└── README.md               # This file
```

### ✅ Submission Guidelines

1. **Naming Convention**: Use lowercase names with hyphens (e.g., `nabeel-nasir`, `john-smith`)
2. **Organization**: Create separate folders for each assignment or project
3. **File Structure**: Keep your work organized and include relevant files
4. **Documentation**: Add a brief README or comments explaining your work when necessary
5. **Git Commits**: Use clear, descriptive commit messages for your submissions

### 📝 Example Structure

```
students/
├── john-smith/
│   ├── assignment-1/
│   │   ├── solution.py
│   │   └── README.md
│   └── assignment-2/
│       ├── app.js
│       └── style.css
└── jane-doe/
    ├── project-1/
    │   ├── src/
    │   ├── tests/
    │   └── README.md
```

### 🚫 What NOT to Include

- Large binary files or datasets
- Unnecessary dependencies or node_modules
- Personal or sensitive information
- Duplicate files or submissions

### 📌 Important Notes

- Make sure your assignments are properly tested before submission
- Follow the coding standards and guidelines provided in class
- Reach out if you have questions or need clarification
- Keep your folder clean and well-organized
